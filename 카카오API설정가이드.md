# 카카오 API 설정 가이드

## 현재 상태
- ✅ API 키 로드 성공
- ❌ 403 오류 발생 (플랫폼 설정 필요)

## 해결 방법

### 1. 카카오 개발자 콘솔 접속
https://developers.kakao.com/

### 2. 내 애플리케이션 선택
- 로그인 후 "내 애플리케이션" 메뉴
- 생성한 앱 클릭

### 3. 플랫폼 설정
1. 왼쪽 메뉴에서 **"플랫폼"** 클릭
2. **"Web 플랫폼 등록"** 버튼 클릭
3. 사이트 도메인 입력:
   ```
   http://localhost:8501
   ```
4. **저장** 버튼 클릭

### 4. (선택사항) 활성화 설정
1. 왼쪽 메뉴에서 **"앱 설정" → "요약 정보"** 클릭
2. **"상태"**가 **"사용 중"**인지 확인

### 5. 설정 완료 후 테스트
```bash
py test_api.py
```

성공 시 다음과 같은 메시지가 표시됩니다:
```
🎉 API 키가 정상적으로 작동합니다!
```

## REST API 키 확인 방법
1. 카카오 개발자 콘솔 → 내 애플리케이션
2. 왼쪽 메뉴 **"앱 설정" → "앱 키"**
3. **"REST API 키"** 복사
4. `.env` 파일에 붙여넣기:
   ```
   KAKAO_API_KEY=여기에붙여넣기
   ```

## 주의사항
- API 키 앞뒤에 따옴표나 공백 없이 입력
- 환경변수 이름은 정확히 `KAKAO_API_KEY`
- `.env` 파일은 프로젝트 루트 디렉토리에 위치

## 문제 해결
플랫폼 설정 후에도 안 되면:
1. 브라우저 새로고침
2. Streamlit 재시작:
   - 터미널에서 Ctrl+C
   - `py -m streamlit run ex2-1.py` 다시 실행
